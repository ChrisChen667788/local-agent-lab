# v0.2.1 · 2026-03-28

## Scope

- 收口 benchmark 执行链稳态优化，重点压低 full-suite 远端长跑中的 `terminated / aborted` 类失败。

## Included

- Runtime / benchmark updates:
  - 远端 benchmark 默认 timeout 从 `90s` 提高到 `120s`
  - 对 `thinking`、`tool-first`、代码与长任务 workload 进一步放宽 timeout
  - `terminated`、`aborted`、`stream idle timeout` 纳入 remote retry
  - 远端流式 body 读取补了空闲超时保护，避免 benchmark 长跑无响应悬挂
  - failed 摘要语言统一到 `进度卡 / 结果汇总 / Benchmark 历史`
  - failed 类型统一成中文标签，并补充 hover 解释
  - failed 热点分布补到后台，支持按 `target / profile / workload` 快速定位
- Admin UI updates:
  - 交叉热力图明确显示当前评测对象
  - 热力图格子新增“推荐用途”提示
  - `n=0` 的格子明确显示 `暂无样本`
  - 曲线图区块上移到 `Benchmark 进度` 下方，便于先看执行态再看趋势
