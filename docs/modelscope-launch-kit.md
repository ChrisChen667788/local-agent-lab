# ModelScope Launch Kit · First LLM Studio

## Recommended repository name

- English: `First LLM Studio`
- Chinese: `First LLM Studio｜本地优先 LLM 工作台`

## Recommended short description

### English

A local-first LLM studio for Apple Silicon with MLX local runtimes, remote API comparison, benchmark ops, Compare Lab, replay, trace review, and runtime recovery.

### 中文

面向 Apple Silicon 的本地优先 LLM 工作台，集成 MLX 本地运行时、远端 API 对比、benchmark 运维、Compare Lab、replay、trace review 和 runtime recovery。

## Target users

### English

- local AI builders on Apple Silicon
- agent and tooling teams
- evaluation and platform engineers

### 中文

- Apple Silicon 本地 AI 开发者
- Agent / 工具链团队
- 评测 / 平台工程团队

## Core value bullets

### English

- compare local and remote models under fair constraints
- keep experiments, benchmark, and runtime operations in one workspace
- inspect replay, trace, tool calls, and recovery actions with evidence
- monitor local gateway hardware cost while iterating

### 中文

- 在统一约束下对比本地与远端模型
- 把实验、benchmark 与 runtime 运维收在同一个工作台里
- 用 replay、trace、tool call 和恢复动作做证据链审阅
- 在迭代时同步观察本地网关硬件开销

### 中文

### English

First LLM Studio is the workspace I wanted for running MLX local models, comparing them against remote APIs, reviewing benchmark regressions, and diagnosing runtime issues without splitting the workflow across separate tools.

### 中文

First LLM Studio 是我想要的那种工作台：把 MLX 本地模型、远端 API 对比、benchmark 回归审阅和 runtime 排障统一放在一个工作流里，而不是拆在多个工具之间。
